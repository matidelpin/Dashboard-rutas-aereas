# Dashboard de Análisis de Rutas Aéreas ✈️

## Descripción
Proyecto de Business Intelligence desarrollado en Excel Avanzado. Analicé una base de 200 registros de vuelos para identificar oportunidades de rentabilidad y demanda de diferentes aerolíneas y destinos.

## Preguntas de Negocio Resueltas
1.  **¿Cuál es la ruta más demandada?**
2.  **¿Cuál es la ruta más rentable?**
3.  **¿Qué aerolínea factura más?**
4.  **¿Cuál es la época del año con mayor demanda?**
5.  **¿Qué destino tiene mayor demanda?**
6.  **¿Qué aerolínea mueve más pasajeros?**

## Herramientas Usadas
- **Excel Avanzado**: Tablas dinámicas, gráficos dinámicos
- **Análisis de datos**: Limpieza y modelamiento de datos

## Resultados Clave
- Identifiqué que la ruta con mayor demanda es Dubai-Roma.
- La demanda se incrementa un 63% entre Enero y Julio.
- FastJet es la compañía aérea con la mayor participación de mercado, ya que concentra cerca del 20% de los ingresos totales del sector.
